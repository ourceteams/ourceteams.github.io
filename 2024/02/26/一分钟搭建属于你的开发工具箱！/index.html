一分钟搭建属于你的开发工具箱！

永恒君
发布日期:   2024-02-26

大家好，我是热爱开源的了不起！

作为一名程序员，每天工作中都需要用到各种各样的工具，比如 JSON 格式化、正则表达式测试、二维码生成等等。这些工具虽然都很小，但是却非常实用。了不起觉得，如果能够把这些工具整合在一起，那就方便多了。

于是，了不起找到了 it-tools 这个开源项目。

简介

it-tools 是一個面向开发人员和IT专业人员的在线工具集合，采用 Vue.js（Vue 3）和 Naive UI 组件库开发。支持Docker私有化部署，包含了各种实用的开发工具、网络工具、图片视频工具、数学工具等等，具有功能全面、部署方便、使用简单等特点。

目前在github上收获了8.1K的star。

安装部署

官方推荐使用docker进行安装部署，非常简单方便。这里提供了两个镜像，根据情况选一个就行。

docker hub:

github packages:

使用体验

这个就是项目的首页，很简洁。

目前已经支持中文在内的多种语言。

支持深浅两种主题，可根据自身情况选择。

我们来看看都有哪些工具，主要是下面这几类：加密、转换器、web、图片和视频、开发、网络等等。

选几个给大家看看。

加密工具

包括 Token生成器，Hash文本，UUIDs生成器、加密/解密文本等。

转换器

各种转换工具，有日期时间转换、Base64文件转换、文本到ASCII二进制、文本转Unicode等。

web工具

有编码/解码url格式的字符串、转义html实体、Url分析器等

开发工具

有Crontab表达式生成、Json压缩、Json美化和格式化、Json转CSV、SQL美化和格式化

后面还有很多，如果你不想按类别找，也可以按ctrl+k，通过关键词来进行搜索，比如json

目前大致有约100个常用工具，基本可以满足日常的开发使用。

项目迭代更新非常频繁，工具后面还会不断增加，功能将会丰富，如果有需求的话也可以去提你的issue。

总的来说，it-tools 是一个非常实用的开源项目，值得各位小伙伴们收藏使用。一定能帮你提高工作效率，节省时间和精力。

更多项目细节和功能，感兴趣的小伙伴自行去项目地址探索~~
