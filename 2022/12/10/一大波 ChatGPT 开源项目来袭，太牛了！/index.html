没想到，ChatGPT 一手太极打得也是可以。

整体体验下来，ChatGPT 还是蛮超出预期的，给出答案也是基本准确，语言文字组织地也非常的不错。

这么火爆、优质的项目，怎么能少的了 GitHub 上交友者的开发热情呢？小妹来带大家看看，他们都是如何与 ChatGPT 进行深入连接的？

浏览器插件

ChatGPT for Google

这是一个 Chrome 插件，安装插件之后，在你 Google 搜索内容的时候，除了会显示正常的结果，右侧还会同时展示 ChatGPT 的结果，可以相互参考，非常的贴心。

插件除了支持 Google 之外，Bing, DuckDuckGo, Brave, Yahoo, Naver, Yandex, Kagi等等也是支持的。

ChatGPT Chrome Extension

同样是一款 Chrome 插件。这款插件更加直接，安装好后，

打开网页的文本框，输入问题后右键，点弹出的「Ask ChatGPT」的选项，ChatGPT 会根据你写的内容，生成相应的结果。非常方便快捷，很适合改代码、写短文、邮件等。

微信聊天

ChatGPT 这么强大，自然会想到与微信搭配使用，岂不是完美？

WeChat GPT

这个项目基于 wechaty 配合 Docker 一起使用，可以实现通过微信聊天窗口，发起与 ChatGPT 的对话。

使用之前需要配置 OpenAI 的账号信息，支持群聊和私聊，支持多个 Account 混合使用。

ChatGPT WeChat Bot

这个项目和上面项目一样，基于 webchaty 开发，配合 Docker 使用。

配置好用户信息后，就可以开始自由的“调戏了”，这是实际使用效果：

还支持群机器人，@它并发起一个问题，便能得到响应。

API开发

这是来自一位来自马来西亚的开发者 Antonio Cheong的开源，他通过对 ChatGPT 进行逆向，提取并制作成了 API。

有了 API，就方便大家开发各种有趣应用了，比如代码辅助、智能聊天等应用。

Mac 插件 - ChatGPT for desktop

这是一款专门为 Mac 用户制作的工具，安装好后，按 Cmd+Shift+G 快捷键，可以在系统菜单栏快速启动 ChatGPT。

有网友发现，这个工具其实也支持 Win 系统，需要运行 npm install electron-forge 命令自行编译。

Node.js API 接口

如果你平时习惯用 Node.js 开发，那么小妹建议你可以关注下「ChatGPT API」这个项目。

这个项目对 ChatGPT 的 API 进行了二次封装，让你可以更加轻松的进行各种有趣开发。

通过 npm 进行安装即可快速使用。

小结

总的来说，ChatGPT 是小妹今年看到的最有趣的 AI 项目了，她已一种更为贴地气的方式，让普通人都近距离亲身体验到当下先进的 AI 技术的成果。

最关键的是，相比之前的 AI ，ChatGPT 解决问题的满意率是比较高的。对于普通人来说，可以很方便的使用这个项目，比较精准的解决日常当中一些诸如写稿、搭框架、改代码等等一些很费脑力的工作。

当然也不是很完美，但小妹觉得 ChatGPT 目前已经很优秀了，各位觉得呢？
