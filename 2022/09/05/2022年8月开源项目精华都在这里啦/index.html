2022年8月开源项目精华都在这里啦~

永恒君
发布日期:   2022-09-05

大家好，我是可爱又机灵的开源小妹!

小妹花了点时间整理了一下2022年8月分享过的酷酷项目，方便大家进行查看和收藏！

1、RuoYi：一个后台管理系统，基于 Spring Boot、Apache Shiro、MyBatis、Thymeleaf，目的让开发者注重专注业务，降低技术难度，从而节省人力成本，缩短项目周期，提高软件安全质量。

查看文章

2、PathPlanning：实现了一些常见路径规划算法，包括基于搜索的算法和基于采样的算法，并为每个算法设计了动画进行展示运行过程。

查看文章

3、Mall4j：一个完整的开源的电商系统，采用现阶段流行技术实现。
项目分为四个主要项目，java后台、vue后台前端、小程序、uni-app。代码风格良好，文档完善，社区氛围也不错，作者维护也很及时。。

查看文章

4、Dendron：一个开源的、笔记可本地化的、基于 Markdown 语法记录的笔记软件，可以与 VSCode 等 IDE 原生集成，非常适合开发人员构建自己的个人知识体系！

查看文章

5、AJ-Report：一个可视化拖拽编辑的、直观酷炫，具有科技感的图表工具开源项目。

查看文章

6、WinMerge：一个免费开源的文件内容差异对比软件，支持文本文件、文件夹、图片、二进制等等文件比较和表格比较，非常适合程序员或者经常写文档、整理资料的人使用。

查看文章

7、daedalOS：一款开源的基于WebAssembly在浏览器中实现桌面操作系统，能够让你直接在浏览器内运行 Win 系统，而且是非常丝滑般的体验。

查看文章

8、Tduck 填鸭收集器：一款开源的表单在线收集系统，功能强大，界面美观。

查看文章

9、Jeepay：一套适合互联网企业使用的开源支付系统,目前有交易、退款、转账、分账等接口，对接微信,支付宝,云闪付官方接口,支持聚合码支付。

查看文章

10、magic-api：一款基于 Java 的可视化 HTTP API 接口快速开发框架，通过 UI 界面编写接口，无需定义 Controller、Service、Dao 等 Java 对象就能完成常见的接口开发。

查看文章

11、code996：一个代码分析工具，它将 Git 项目中的 commit 时间分布进行了可视化的展示，进而可以轻松的看出编码工作的实际强度。

查看文章

12、YYEVA（YY Effect Video Animate）：YY直播团队开源的移动端适配动画解决方案，该方案包含了客户端渲染引擎，设计配套工具等。

查看文章
