18.6K Star！这款 JSON 数据可视化工具太优雅了！

永恒君
发布日期:   2022-10-27

大家好，我是可爱又机灵的开源小妹！

最近听到说2023年国考要开始，今年这个形势啊，估计又会是卷上天的节奏了，真的好累啊~~~

话说有打算去试试手气的小伙伴吗，评论区里来说说你的故事吧~~

言归正传，平时出于各种原因，好多小伙伴都喜欢 API、数据库等的 JSON 格式。
有时候一些 JSON 请求或响应很简单，只包含几个对象。但是在大多数情况下，JSON 结果都会是出乎你的意料，更可怕的是，出于各种原因和分析目的，还需要进一步对进行 JSON 深入研究。各位小伙伴们一定都深有体会~

最近小妹就偶遇到了一个非常优雅的 JSON 数据可视化工具 - jsoncrack，可以一眼看出 JSON 的奥妙，简直爱的不要不要的~~

简介

jsoncrack - 一个开源的在线 JSON 数据可视化编辑器，可以把任意JSON数据转化为树状图。

目前在收获了 18.6k Star，足见该项目的受欢迎程度。

这个工具最大的特色就是，不仅格式化数据了，还帮你生成了可视化的思维导图的形式，看起数据来更立体。

支持放大/缩小、展开/收缩、搜索节点、导出图片等操作，还可以快速部署成服务。

使用体验

可视化

打开工具主页，点击 “GO TO EDITOR”进入编辑器，你会看到一个作者准备好的 JSON 示例。

清空左边的代码，将你需要可视化的 JSON 代码粘贴进去即可将 JSON 可视化，非常简单。

接下来我们来深入看看 jsoncrack 提供的其他功能。

放大/缩小

可点击右上角的+/-号进行图像的放大/缩小，直接用鼠标滚轮也是可以的。

保存导图

可以直接选择喜欢的背景颜色，进行图片的保存。

这是刚刚保存的图片

旋转导图

如果你想更换一下思维导图的方向，同样也是支持的。

压缩/恢复结点

折叠/展开结点

搜索结点

另外还有导入、保存（下载）、分享以及节点、主题等等的设置。

怎么样，用这个工具来展示 JSON 是不是太优雅了？小妹太爱作者了~~
